# Zenix Session Start

Always read before making changes:

1. `README.txt`
2. `PROJECT_MEMORY.md`

Required defaults for this project:
- Mobile startup splash must not block app display.
- Toast notifications are desktop-only and bottom-right.
- Prioritize FR playback (`VF`, `VOSTFR`, `MULTI`) before `VO`.
- Keep auto source switching active when playback blocks.

Deployment checklist:
1. Commit changes.
2. Push `origin main`.
3. Push `deploypublic main`.
4. Verify `https://zenix.best` serves latest asset version.
